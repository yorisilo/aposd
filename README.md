# 輪読会の方法
- 事前準備は基本的にはしない
- 集まったタイミングで本を一緒に読みメモを書いていく
- ある程度読み終わったら、お互いに読んだ内容について議論していく
- 議論の最中にもメモを行う
- 終わったら、次に読む部分を決める

# 具体例
- 30分お互いに読み進めて、メモをする
- 30分議論する。議論中もメモを書いていく
  - メモはこちらで行う予定: https://github.com/yorisilo/aposd/discussions/2
